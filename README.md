mcmmo-groups
============

A small bukkit plugin to allow players to join groups (Guilds / Towns / etc) and gain a bonus to a few types of experience gain depending on the group.


Dependencies
------------

Requires mcmmo and Vault.


Build Targets
-------------

Craftbukkit 1.3.1R2
mcMMO v 1.3.11
Vault 1.2.18
